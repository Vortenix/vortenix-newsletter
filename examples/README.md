# Examples

- [`minimal-vertical.yaml`](minimal-vertical.yaml): a generic Storage vertical.
- [`sample-audience.yaml`](sample-audience.yaml): an audience selecting that vertical.
- [`sample-connector.py`](sample-connector.py): an offline `SourceConnector` implementation.
- [`sample-vertical.py`](sample-vertical.py): a specialised vertical skeleton.
- [`sample-workflow.py`](sample-workflow.py): programmatic offline workflow invocation.

Examples are educational and are not imported by the package. Copy configuration examples into the corresponding `config/` files and validate them before use.
